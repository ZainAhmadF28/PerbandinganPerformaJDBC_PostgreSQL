📋 CHECKLIST SETUP (Ikuti urutan ini!)

□ STEP 1: INSTALL POSTGRESQL
  ├─ Download: https://www.postgresql.org/download/
  ├─ Install dengan default settings
  ├─ Catat password user 'postgres'
  └─ PostgreSQL akan jalan di port 5432

□ STEP 2: BUAT DATABASE
  ├─ Buka pgAdmin 4
  ├─ Klik kanan Databases → Create → Database
  ├─ Nama: db_pegawai
  └─ Klik Save

□ STEP 3: JALANKAN SQL SCRIPT
  ├─ Klik kanan database 'db_pegawai' → Query Tool
  ├─ Buka file: database_setup.sql
  ├─ Copy semua isi (skip baris 5-6 yang di-comment)
  ├─ Paste ke Query Tool
  └─ Klik Execute (▶️) atau tekan F5

□ STEP 4: VERIFIKASI DATABASE
  ├─ Jalankan: SELECT * FROM pegawai;
  ├─ Harus muncul 5 data
  └─ Cek functions dengan query di database_setup.sql baris terakhir

□ STEP 5: DOWNLOAD JDBC DRIVER
  ├─ Buka: https://jdbc.postgresql.org/download/
  ├─ Download: postgresql-42.7.1.jar
  └─ Copy ke folder: lib/

□ STEP 6: IMPORT KE ECLIPSE
  ├─ File → Import → Existing Projects into Workspace
  ├─ Browse ke folder: PerbandinganPerformaJDBC_PostgreSQL
  └─ Klik Finish

□ STEP 7: EDIT PASSWORD
  ├─ Buka: src/com/jdbc/config/DatabaseConfig.java
  ├─ Cari baris: private static final String PASSWORD = "admin";
  └─ Ganti "admin" dengan password PostgreSQL Anda

□ STEP 8: RUN APLIKASI
  ├─ Buka: src/com/jdbc/main/MainApp.java
  ├─ Klik kanan → Run As → Java Application
  └─ Lihat output di Console

❌ TROUBLESHOOTING CEPAT

ERROR: "ClassNotFoundException: org.postgresql.Driver"
FIX:  JDBC Driver belum ditambahkan
      → Klik kanan project → Properties → Java Build Path → Libraries
      → Add External JARs → Pilih postgresql-42.7.1.jar

ERROR: "Connection refused"
FIX:  PostgreSQL tidak jalan
      → Buka Services (Windows) → Start postgresql-x64-xx

ERROR: "password authentication failed"
FIX:  Password salah
      → Edit DatabaseConfig.java → Ganti PASSWORD

ERROR: "database db_pegawai does not exist"
FIX:  Database belum dibuat
      → Ulangi STEP 2

ERROR: "function insert_pegawai does not exist"
FIX:  Functions belum dibuat
      → Ulangi STEP 3
